# Amazon 进货选品 — 使用说明

本文档帮助您使用选品系统，在 Amazon 上查找值得进货的商品。

---

## 一、这个工具能帮您做什么

输入想进货的**商品类别或关键词**（例如：杯子填 `cup`，蓝牙耳机填 `wireless earbuds`），系统会自动在 Amazon 美国站搜索同类商品，并列出**卖得最好的前 N 名**，同时给出：

- 商品名称、图片  
- 上月大概卖了多少（Amazon 官方显示的购买热度）  
- 买家评分和评论数量  
- **进货建议**：是否值得进货，以及理由说明  

点击商品名称可打开 Amazon 官网，查看详情和**当前售价**（系统内不显示价格，以官网为准）。

---

## 二、如何打开

在浏览器地址栏输入：

**http://localhost:8000**

（若无法打开，请联系负责维护的同事确认服务是否已启动。）

---

## 三、如何开始一次选品

### 第一步：填写搜索条件

| 填写项 | 说明 | 建议 |
|--------|------|------|
| **商品类别 / 关键词** | 想进货的商品类型，建议用英文 | 如 `cup`、`yoga mat` |
| **品类内取前 N 名** | 最终要看几条推荐 | 一般填 `20` |
| **扫描页数** | 搜索范围大小，页数越多结果越全，等待时间略长 | 建议 `6~10` |
| **补充详情** | 填 `0` 即可（最快）；若需要更详细的类目排名信息，可填 `3~5` | 默认 `0` |
| **评论数** | 仅在上一步「补充详情」大于 0 时有用；填 `0` 表示不额外抓取评论 | 默认 `0` |

### 第二步：点击「开始选品」

- 页面会显示进度，请耐心等待（通常约 **1～3 分钟**，页数多或补充详情时会更长）  
- 完成后会自动展示结果表格和商品卡片  

### 第三步：查看并决策

- 优先看 **「结论」** 列为「强烈推荐进货」「可考虑进货」的商品  
- 阅读每条商品下的 **进货理由** 和 **风险提示**  
- 点击商品名到 Amazon 核对价格、规格、物流等再决定是否进货

## 四、如何看懂结果

### 4.1 对比总览表（表格）

方便一眼对比多个商品：

| 列名 | 含义 |
|------|------|
| **#** | 排名，第 1 名表示在该品类中上月销量表现最好 |
| **商品** | 商品名称（摘要） |
| **上月销量** | Amazon 显示的近期购买热度，如「5K+ bought in past month」表示上月约 5000+ 人购买 |
| **BSR** | 类目销量排名，数字越小通常卖得越好；没有显示时可忽略 |
| **评分** | 买家星级，满分 5 分 |
| **评论** | 评论总数，评论多一般表示市场更成熟 |
| **进货指数** | 0～100 分，分数越高越值得重点考虑 |
| **结论** | 系统给出的进货建议等级 |

### 4.2 结论等级说明

| 结论 | 含义 |
|------|------|
| **强烈推荐进货** | 销量、口碑等综合表现较好，可优先调研 |
| **可考虑进货** | 整体不错，建议结合成本和竞争再决定 |
| **谨慎观察** | 存在明显短板，需额外核实 |
| **不建议进货** | 综合表现偏弱，不建议作为首选 |

### 4.3 商品卡片（详细说明）

每张卡片包含：

- **彩色标签**：进货结论  
- **进货理由**（绿色 ✅）：如销量靠前、口碑好、竞争适中等  
- **风险提示**（红色 ⚠️）：如评论过多可能已是红海、评分偏低等  
- **商品链接**：跳转 Amazon 查看详情与价格

## 五、历史报告

页面底部 **「历史报告」** 保存了之前做过的选品记录。

| 操作 | 说明 |
|------|------|
| **查看** | 在页面重新打开该次选品结果 |
| **删除** | 删除该条记录（删除前会提示确认，删除后无法恢复） |

## 六、使用建议

1. **关键词尽量准确**：越贴近您想进的品类，结果越有用（如进马克杯可试 `coffee mug`，不要只写 `cup` 若结果太杂可换词）。  
2. **扫描页数不要太少**：只扫 1～2 页时，对比样本少，排名参考价值有限；建议 **6 页以上**。  
3. **以官网为准**：价格、库存、规格、运费等务必在 Amazon 商品页再次确认。  
4. **结合自己的成本**：系统只提供市场参考，最终是否进货还需算清楚采购价、物流、平台费用等。  
5. **同一品类可隔段时间再查**：市场变化快，可定期重新选品对比。

## 七、常见问题

**问：为什么页面上没有价格？**  
答：Amazon 上价格会变，且不同规格价格不同。请点进商品链接，在官网查看准确售价。

**问：等很久还没出结果？**  
答：扫描页数多或网络较慢时会久一些。若长时间无反应，可刷新页面后重试，或联系维护人员。

**问：很多商品「上月销量」是空的？**  
答：部分商品 Amazon 不展示该信息，属于正常情况，可重点看有数据的商品。

**问：「强烈推荐」是不是一定能赚钱？**  
答：不是。系统根据公开信息做参考排序，不保证利润，进货前请自行核算。

**问：重启电脑或关闭服务后，历史报告还在吗？**  
答：已保存的历史报告一般仍在；若服务重新部署，以实际页面列表为准。

**问：可以搜中国亚马逊吗？**  
答：当前默认是美国站（Amazon.com），搜英文关键词效果最佳。

## 八、注意事项

- 本工具仅供 **内部选品参考**，请勿用于对外公开传播爬取的数据。  
- 选品结果基于 Amazon 公开展示信息，可能与实际情况有偏差。  
- 请合理、低频使用，避免短时间内反复大量搜索。
